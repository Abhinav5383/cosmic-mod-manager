QUERY PARAMETERS​

  • query
    key: q
    type: string
    Example: q=zoom

  • sortby
    key: sortby
    type: ENUM { relevance | downloads | follow_count | recently_updated | recently_published }
    default: relevance
    Example: sortby=downloads

  • offset
    key: offset
    type: number
    Example: offset=20

  • limit
    key: limit
    type: number
    default: 20
    max: 100
    Example: limit=50
